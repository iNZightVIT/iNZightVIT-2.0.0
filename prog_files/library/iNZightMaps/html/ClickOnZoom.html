ClickOnZoom {iNZightMaps}R Documentation

Zoom in/out

Description

Zoom in/out when click the plot

Usage

ClickOnZoom(ratio = 1, resize = FALSE,
  p.center = global.objects$maps$pf$click.points)

Arguments

ratio

a numeric value, define the ratio of zomm in or out

Details

if ratio < 1 then zoom in, if ratio > 1 then zoom out, if ratio = 1 then shift the plot.
