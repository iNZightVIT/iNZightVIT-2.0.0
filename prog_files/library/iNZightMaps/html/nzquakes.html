nzquakes {iNZightMaps}R Documentation

Earthquakes in New Zealand

Description

A dataset containing locations and measurements of earthquakes in the New Zealand region.

Usage

nzquakes

Format

A data frame with 334 rows and 12 variables:

Order

For ordering the observations

Latitude

The latitude ("Y") positions of the earthquakes

Longitude

The longitude ("X") positions of the earthquakes
