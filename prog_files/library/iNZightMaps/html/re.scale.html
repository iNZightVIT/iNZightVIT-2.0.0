re.scale {iNZightMaps}R Documentation

rearrange the limit

Description

rearrange the limit by given a ratio.

Usage

re.scale(x, ratio)

Arguments

x

a vector of length 2.

ratio

a numeric value.

Value

a vector of length of 2, re-sized by the ratio.

Author(s)

Jason Wen

Examples

x = c(0,100)
ratio = 0.5
re.scale(x,ratio)
