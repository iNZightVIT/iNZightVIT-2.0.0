Description

Convert sf* object to an grid graphics object (grob)

Usage

st_as_grob(x, ..., units = "native")

Arguments

x

object to be converted into an object class grob

...

passed on to the xxxGrob function, e.g. gp = gpar(col = 'red')

units

units; see unit
